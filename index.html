关于我

我是 2016 本科毕业的，毕业之前我有考研，但是很遗憾，辛苦认认真真备战一年还是没考上。在本科的期间，我就爱折腾，不曾减少对计算机和网络的世界的好奇，对很多事物 有所接触，但都是知道皮毛，没有精通的编程技术，快要毕业后，才慢慢认清自己的技能，也开始逐渐认识职场、社会。本科毕业后，有幸在当地一家小公司从事 Java 后端开发。

踏入公司，我才开始算是真正走上编程之路，也算是真正踏入编程之门吧。也是毕业之后，我才开始意识到能大佬、牛人他们的厉害的地方，我需要像他们学习。我开始学习 Git， 没事打开 GitHub 逛逛，在 CSDN 写博客，学如何 FQ 去善用谷歌...就这样，我感觉到那是我技术成长最快的时间段。在职业理想方面一直很想去大厂，但觉得自身能力和理想相差太远， 所以在后来，我选择了二战考研，想拥有一个好的平台，但更重要是想沉淀自己。不过遗憾，二战还是以失败告终了。最后，经过很多纠结和矛盾，选择了调剂。 目前就读于江西师范大学计算机信息工程学院，软件工程专硕。2020 届毕业生。

我喜欢这个行业，愿意在这个行业努力，期待未来能有我的一片天地，但我也要说技术不是全部，技术是我热爱的，是我主业，我期望着，能学好技术，做一名优秀的软件工程师， 但我不想技术成为生活的全部，我希望过好生活，多去尝试自己喜欢的，多去品味世界奥妙，让人生丰富多彩。
